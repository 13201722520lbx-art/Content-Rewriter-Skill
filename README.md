# Content-Rewriter-Skill

一个专业的中文内容深度重写与反AI检测技能，适用于支持自定义Skill的AI平台。

## 功能特点

- **深度重写**：不是简单的同义词替换，而是从观点、结构、表达方式全方位重构内容
- **反AI检测**：针对维普等主流AIGC检测系统的检测机制，采用六大策略确保通过检测
- **保持信息完整**：在改写过程中确保原文核心信息不丢失
- **多轮优化**：支持根据反馈持续迭代，直到满意为止

## 适用场景

- 学术论文降重与反AI检测
- 自媒体文章原创度提升
- 内容平台原创审核通过
- 文章风格转换与优化

## 支持的AI平台

### ✅ 完全支持

1. **Claude Code (Anthropic)**
   - 官方CLI工具
   - VS Code / JetBrains插件
   - claude.ai/code 网页版

2. **Cline (原Claude Dev)**
   - VS Code插件
   - 支持自定义Skill功能

### 🔄 可能支持（需自行测试）

以下中国大模型平台如果支持自定义提示词或Skill功能，理论上可以使用：

1. **通义千问 (Qwen)**
   - 如果使用API或支持自定义System Prompt的客户端

2. **文心一言 (ERNIE)**
   - 通过API调用时可以设置System角色

3. **智谱清言 (ChatGLM)**
   - GLM-4等模型支持System Prompt

4. **Kimi (月之暗面)**
   - 如果使用API或支持自定义提示词的工具

5. **豆包 (字节跳动)**
   - 通过API可以设置System消息

6. **讯飞星火**
   - 支持System角色设置

### 📝 使用方法

**对于支持System Prompt的平台**：
将`content-rewriter.md`文件中`---`分隔符之后的所有内容复制到System Prompt或系统提示词设置中。

## 安装方法

### Claude Code / Cline 用户

1. 找到你的`.claude/skills`目录：
   - Windows: `C:\Users\你的用户名\.claude\skills\`
   - macOS/Linux: `~/.claude/skills/`

2. 下载`content-rewriter.md`文件到该目录

3. 重启Claude Code或重新加载Cline

4. 使用命令触发：
   ```
   /content-rewriter
   ```

### 其他AI平台用户

1. 下载`content-rewriter.md`文件

2. 打开文件，复制`---`分隔符之后的所有内容

3. 在你使用的AI平台中：
   - 如果有"System Prompt"或"系统提示词"设置，粘贴到那里
   - 如果没有，可以在每次对话开始时先发送这段内容，然后再发送你的重写需求

## 核心技术

本Skill采用六大反AIGC检测策略：

1. **提升困惑度** - 使用更自然、不那么"完美"的表达
2. **制造突发性** - 长短句交替，节奏变化
3. **打破语义模式** - 避免AI高频过渡词
4. **增加词汇多样性** - 同一概念多种表述
5. **句法结构变异** - 混合使用多种句式
6. **注入人格化特征** - 加入主观视角和情感色彩

## 工作流程

1. 接收原始文章并确认需求
2. 深度分析文章结构和AI痕迹
3. 提出重写方案供确认
4. 执行深度重写
5. 根据反馈循环优化
